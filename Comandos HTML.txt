<nav> É uma seção de uma página que contém links de navegação, ou seja, que apontam para outras páginas ou para outras áreas da mesma página

-----

<ul> Representa uma lista de itens desordenada, onde os itens não têm uma ordem rígida e as suas posições são irrelevantes

-----

<ol> É uma lista ordenada com uma contagem numérica, de letras, algarismos romanos

-----

<li> É usado para representar um item que faz parte de uma lista

-----

<header> É usado como cabeçalho da página

-----

<div> É uma espécie de container que não representa nada visual, mas é utilizado para formatar o conteúdo dentro dessa tag no CSS

-----
